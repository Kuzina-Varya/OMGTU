# РГР: Грузовые перевозки

## Описание

Проект реализует базу данных для предметной области «Грузовые перевозки».

База данных хранит информацию о маршрутах, водителях, перевозках, премиях и коэффициентах оплаты по стажу.

## Используемые технологии

- Python
- PostgreSQL
- SQLAlchemy ORM
- Jupyter Notebook
- Pandas

## Основные возможности

- подключение к PostgreSQL;
- создание схемы базы данных через SQLAlchemy ORM;
- выполнение CRUD-операций;
- выполнение сложных SQL-запросов;
- создание представления;
- создание хранимой функции;
- создание хранимой процедуры;
- создание триггера;
- обработка исключений.

## Запуск
Запустить венв:
```bash
.\.venv\Scripts\Activate.ps1
```

Установить зависимости:
```bash
pip install -r requirements.txt
```

Запустить Jupyter Notebook:
```bash
python -m jupyter lab
```
